# Royalty-Free Music Sources for Mood-Matched Audio

## Best Options:

1. **Pixabay Music** (pixabay.com/music/)
   - 100% free, no attribution required for most tracks
   - Has mood/genre tags
   - Direct download links available
   - API available

2. **Free Music Archive** (freemusicarchive.org)
   - CC0 and CC-BY licensed
   - Searchable by mood/genre
   - API access available

3. **YouTube Audio Library** (youtube.com/audiolibrary)
   - Free to use
   - Searchable by mood
   - Direct downloads

4. **Incompetech** (incompetech.com)
   - Kevin MacLeod's royalty-free music
   - Various moods and genres
   - CC-BY license

## Recommendation:
Use **Pixabay Music API** - it's the easiest to automate and has good mood tagging.
