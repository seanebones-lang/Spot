# Deployment Optimization Prompt Templates

## For GitHub Agent

```
@Eleven MCP /agent github optimize workflow [workflow-name]

Analyze and optimize the GitHub Actions workflow for:
- Performance (parallel jobs, caching)
- Error handling
- Security best practices
- Workflow reusability
```

## For Deployment Agent

```
@Eleven MCP /agent deployment optimize Railway Vercel

Optimize deployment pipelines for:
- Error-free deployments
- Parallel deployment strategies
- Rollback capabilities
- Health check verification
```

## For CI/CD Agent

```
Create comprehensive CI/CD pipeline with:
- Pre-deployment validation

```
